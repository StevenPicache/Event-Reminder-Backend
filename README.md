# Event-Reminder-Backend

## Purpose

To show an understanding of Node, Express, Docker, and Sequelize

## Tech Stack

**Server:**
Node,
TypeScript,
Express,
PostgreSQL,
Docker

## Setup & Running

```bash
  cd Event-Reminder-Backend
  npm install
  docker compose create
  docker compose start
  npm run start
```
